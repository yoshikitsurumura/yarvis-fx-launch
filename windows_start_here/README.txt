このフォルダの中のファイルをダブルクリックしてください。

1) Set_Form_URL.cmd
   - 最初にこれを実行して、フォームのURLを設定します。

2) Run_Server.cmd
   - 画面表示が綺麗になる方法です。動かしたままにしてください。

3) Open_LP_v1.cmd / Open_LP_v2.cmd
   - お問い合わせページ（LP）を開きます。v1/v2で見出しが違います。

4) Open_Demo.cmd
   - 体験画面（やーびす）を開きます。

メモ:
- フォーム設定は lp\config.js に保存されます。
- サーバを止めるには、黒い画面で Ctrl + C を押してください。
